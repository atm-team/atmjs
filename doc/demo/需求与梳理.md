# 站点需求
开发一个博客网站，具有以下功能：
* 网站前台：文章展示,帮助中心
* 用户后台：文章管理功能,分类管理,登陆注册,账号管理等功能
* 管理员后台：用户管理等一系列功能

# 功能梳理及组织结构
```
|--blog(站点)
    |--www(前台项目)
        |--common(前台项目公共模块)
            |--1.0.0
                |--src
                    |--common.js(前台项目的公共脚本)
                    |--common.css(前台项目的公共样式)
```
